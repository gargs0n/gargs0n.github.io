# Source code for [thegarg.de](http://www.thegarg.de)
## Build this project

### Install jekyll

Ensure `jekyll` is installed:

```
gem install jekyll
```

### Install dependencies

Install other dev-dependencies, run this from the root folder:

```
npm install
```

### Start local server

Simply run:

```
npm start
```

Jekyll will serve from `localhost:4000`
